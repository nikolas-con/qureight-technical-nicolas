# OVERVIEW
Frontend: React.js and Redux with parcel-bundler and @babel

Backend: Node.js and express with @babel

Database: Firestore

# SETUP

To install the dependencies :

npm run bootstrap

To run the app in development:

npm run dev

To build the app:

npm run build

To run the app in production locally:

npm run prod_l

To run the app in production:

npm run prod
